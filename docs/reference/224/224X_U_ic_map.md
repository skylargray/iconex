# Lexicon Model 224X — U-Designated ICs by Assembly

Abbreviated companion to the Chapter 7 parts list: one table per assembly, one row per **U** reference designator, exploded from the part-number reference ranges in the full list and sorted numerically. Columns are the U designator, manufacturer/Lexicon part number, parts-bin category, and description.

**Scope** — ICs only: categories ending in "IC", delay-line **Modules**, and the DL-1414 intelligent-display ICs. Non-IC parts that happen to sit at a U position (bridge rectifiers, transistor sockets, IC sockets, connectors, plain LEDs) are excluded. See notes for the inclusion rule and for assemblies whose U map carries caveats.

## Notes

- **Inclusion rule.** A part appears here if its parts-list category ends in "IC", or is `Modules` (delay-line module), or is a DL-1414 4-character display (a U-designated display IC). IC *sockets* (category `Sockets`, e.g. 520-xxxx) are not listed even though they share the U number; the same is true of U-positioned bridge rectifiers (e.g. PS#1 U1 `DIODE,BRIDGE`) and transistor sockets (e.g. PS#3 U3 `CONN,XISTOR`).
- **Ranges exploded literally.** Reference ranges/lists from the full list (`U20-35`, `U1-4,9-11`, `U14,26-28,40,41,50-53`, …) are expanded to one row per position. Within a single board no two ICs share a U number; the only overlaps are in the multi-board retrofit kit:
  - **7.28 LARC ROM/RAM Kit — overlaps are expected.** This is a multi-board retrofit *kit*, not a single board. Its U designators are destination-board-relative, so the DRAM block `U20-35` overlaps the EPROM `U23,24,25` and the 8251A `U22`, which land on different host boards. Treat the kit table as an inventory keyed by destination-board U, not a single-board map.
- **7.5 Data Memory (DMEM)** carries the 16× 4164 DRAM at `U20-35` (exactly 16 positions) plus the 5-tap delay-line module at U59. **7.8 Memory Expansion** is a separate battery-backed board (4× H6514 SRAM at U5,6,12,13; 6× 2732 EPROM at U1-4,9,10) with no DRAM — there is no U14 conflict (U14 there is the 4011).
- **7.13 Power Supply Hardware** is a kit of regulators that mount on the PS#2/PS#3 boards; its U designators (U1–U3) reference those boards, noted inline. Together with PS#2 U4 (LM301), the PS#2 regulator complement is U1,U2 (LM317), U3 (7912), U4 (LM301).
- **7.7 ARU** now lists the full datapath (74F283 adders ×10, 74LS670 register files ×4, 74F157 muxes ×5, 74F374 latches ×4, 74S86 XOR ×5, 74LS194 shift registers ×6, 74LS163 counters ×5, 74LS00 glue ×10, plus 74S175 and 74S04), so this U map is the principal single-board reference for the arithmetic unit.
- **Excluded assemblies** (no U-designated ICs under the rule above): 7.9, 7.10, 7.14, 7.15, 7.16, 7.17, 7.18, 7.19, 7.20, 7.21, 7.22, 7.24, 7.25, 7.27, 7.29, 7.33, 7.34, 7.35.
