# Lexicon 480L — Default Parameters with Decoded Names

Cross-reference of the ROM1 default-parameter records (the 144-byte program records) against the parameter labels in **ROM4's page tables** and the **per-program preset values printed in the owner's manual**. The standard-reverb record layout has been solved and verified; other algorithm families are given with their ROM4 parameter names plus raw values.

## How the mapping was established

The manual prints exact displayed values for every Bank-1 Hall (e.g. *Large Hall: RTM 2.19s, SHP 126, SIZ 37m, DIF 99…*). Correlating those values against each word position in the ROM1 records pins every parameter to a fixed slot — exact matches for the un-scaled parameters (SHAPE, DIFFUSION, DECAY OPT) and perfect rank-correlation for the scaled ones (SIZE, RT MID, BASS MULT, PREDELAY, HF CUTOFF, RT HF CUT). Parameter **names** come from ROM4's page tables (slider line + six descriptors per page) and agree with the manual.

## Verified standard-reverb record map  (algorithms `0x02`, `0x06`)

Applies to Banks 1–4 (Halls, Rooms, Wild Spaces, Plates). Words not listed are structural: `w0` = record/algorithm tag (`0x18`), `w2` = fixed (`0xFF`; `0xFE` for Wild Spaces).

| Word | Parameter | Unit | Encoding |
|---|---|---|---|
| `w1` | SIZE | m | size index (≈raw+7 m) |
| `w3` | RT MID | s | reverb-time index |
| `w4` | BASS MULT | x | ×0.25 → multiplier |
| `w5` | PREDELAY | ms | ×2 → ms |
| `w6` | SPREAD | — | 0–255 |
| `w7` | SHAPE | — | 0–255 direct |
| `w20` | HF CUTOFF | Hz | freq index |
| `w21` | CROSSOVER | Hz | freq index (XOV) |
| `w22` | RT HF CUT | Hz | freq index |
| `w23` | DIFFUSION | — | 0–99 direct |
| `w24` | DECAY OPT | — | 0–7 (REVERB n); 0=R0 |
| `w8`–`w19` | Early reflections | — | six (delay ×2ms, level-code) pairs; zero unless +Stage |

> Confidence: **high/verified** for `0x02`/`0x06`. WET/DRY MIX is not stored per-preset here (presets are full-wet). CROSSOVER (`w21`) is the least-certain slot.

## Algorithm parameter reference (ROM4 page tables)

Every parameter page decoded from ROM4: the six slider abbreviations and their full names/units. Use these names with the raw words for the non-reverb banks below.


**Sampler**
- `[CUE KEY 1 TAP FOR EDIT]` → CUE=HEAD TIM FAD RATE CUE, KEY=HEAD TRIM  2, 1=FWD  TIME  2, TAP=FADE TYPE  2, FOR=RATE 2, EDIT=CUE KEY    2 TAP FOR EDIT
- `[PLAY RP TLV PLAY RP TLV]` → PLAY=PLAY KEY   1 RP     DB, RP=REPEAT     1 PER PLAY, TLV=TRIGGER    1 AT, PLAY=PLAY KEY   2 RP     DB, RP=REPEAT     2 PER PLAY, TLV=TRIGGER    2 AT
- `[MID REF RAT MID REF RAT]` → MID=MIDIPLAY   1 RP     DB, REF=REF MIDNOT 1, RAT=RATE 1, MID=MIDIPLAY   2 RP     DB, REF=REF MIDNOT 2, RAT=RATE 2

**Test oscillator / delay**
- `[DLY RUN KIK FLT ANL L/R]` → DLY=FBK DELAY MS, RUN=RUN FEEDBACK DB, KIK=KICK START DB, FLT=FILTER, ANL=ANAL ON/OFF 0=OFF, L/R=ANALIN L/R 0=L

**Other / utility**
- `[LEV FIN +DC -DC ADL ALV]` → LEV=OUT LEVEL DB, FIN=FINE, +DC=POS DC, -DC=NEG DC, ADL=ANL DELAY MS, ALV=ANL LEVEL DB

**Stereo Adjust / EQ**
- `[LEV FIN BAL LNK FIN FIN]` → LEV=STEREO LVL DB, FIN=FINE LVL DB, BAL=L DB BAL  R, LNK=STEREO LINK, FIN=FINE FREQ L, FIN=FINE FREQ R
- `[LVL FIN BAL ROT BAS TRB]` → LVL=STEREO LVL DB, FIN=FINE LVL DB, BAL=L DB BAL  R, ROT=L DB ROT  R, BAS=L&R BASS EQ, TRB=L&R TREB EQ

**Parametric EQ**
- `[XOV HFC HFC SPC TBL TBR]` → XOV=BASS XOVER HZ, HFC=STREB XOVER HZ, HFC=ITREB XOVER HZ, SPC=SPATIAL EQ, TBL=TREBLE LEFT, TBR=TREBLE RT

**Binaural simulator**
- `[CH1 CH2 EFF ANG LFB MIX]` → CH1=LEFT IN, CH2=RIGHT IN, EFF=EFFECT, ANG=SPEAKR ANGLE DEG, LFB=LF L-R BOOST NORM=0, MIX=DRY INPUT MX

**Standard reverb (Hall/Room/Plate, RHall)**
- `[RTM SHP SPR SIZ HFC PDL]` → RTM=RT MID SEC, SHP=SHAPE, SPR=SPREAD, SIZ=SIZE METERS, HFC=HF CUTOFF HZ, PDL=PREDELAY MSEC

**Other / utility**

**Compressor**
- `[DLY SLP GAIN THR ATC RTC]` → DLY=FRONT    DLY MS, SLP=COMP RATIO .  :1, GAIN=MAX GAIN DB, THR=THRESHOLD DN - DB, ATC=ATTACK TC MS, RTC=RELEASE TC MS

**Other / utility**
